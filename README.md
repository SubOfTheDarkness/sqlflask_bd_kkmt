# Ну это короче типо шаблон для проекта по БД
Юзаем мы Flask и SqlAlchemy.
## Установка
Для начала нужно вообще питон поставить, но это я думаю разберетесь.
Потом в терминале(powershell/cmd/bash/терминал редактора) пишем:
```
pip install --upgrade pip
pip install --upgrade sqlalchemy flask
```
Это все библы поставит нужные.
Вообще по хорошему бы виртуальные окружения использовать, но мне впдалу объяснять как и че с ними делать.
