# Инфо
Проект по МДК 11.01 "Технология разработки и защиты баз данных" ККМТ ТУ им. Леонова.
Выполнили студенты 2 курса группы П1-24 Капилов Д. и Агарков Д.
## Установка
Необходимые библиотеки указаны в requirements.txt. Для их установки достаточно запустить:
```
pip install -r requirements.txt
```
Рекомендуется пользоваться pip в [виртуальных окружениях](https://docs.python.org/3/library/venv.html)
## Запуск
Сервер запускается файлом run.py. Сайт находится на [localhost:5000](https://localhost:5000)
