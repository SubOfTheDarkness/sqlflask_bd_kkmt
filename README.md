## Установка
Необходимые библиотеки указаны в requirements.txt. Для их установки достаточно запустить:
```
pip install -r requirements.txt
```
Рекомендуется пользоваться pip в [виртуальных окружениях](https://docs.python.org/3/library/venv.html)
## Запуск
Сервер запускается файлом run.py. Сайт находится на [localhost:5000](https://localhost:5000)
### Warn
Из-за своеобразной работы Google SMTP(отправка сообщений по почте), есть возможность зарегаться с несуществующей почтой, и это не будет ошибкой, т.к. gmail отправит письмо, а потом придёт ответное о том, что такой почты не существует. Перехват этого самого ответного входящего письма реализовать можно, но сложно, и только с костылями, поэтому просто не получится подвердить эту почту, соответственно и залогиниться. <br>
Юзеры, не подвердившие свою почту в течении часа, автоматически удаляются из БД.
